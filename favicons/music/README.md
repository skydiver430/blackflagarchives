# Music category favicons

Add these icons locally (do not use external URLs in production):

- **cobalt.png** — from https://cobalt.tools/icons/apple-touch-icon.png
- **doubledouble.png** — from https://us.doubledouble.top/record.png
- **ncs.png** — favicon from https://ncs.io/ (e.g. https://ncs.io/favicon.ico or site icon)
- **youtube.png** — used by the Tools filter pill (YouTube) in js/app.js; 16×16.
- **stacher.png** — from https://t0.gstatic.com/faviconV2?client=SOCIAL&type=FAVICON&fallback_opts=TYPE,SIZE,URL&url=https://stacher.io&size=48
- **ytmp3.png** — from https://app.ytmp3.as/apple-touch-icon.png

Save each under this directory and reference as `favicons/music/<name>.png` in `data/resources.json` or app.js as appropriate.

**Shared (site root favicons):** For the Discord corner link on tiles, add `favicons/discord.png` (Discord logo; e.g. from [Discord CDN](https://cdn.prod.website-files.com/6257adef93867e50d84d30e2/62fddf0fde45a8baedcc7ee5_847541504914fd33810e70a0ea73177e%20(2)-1.png) or use an official Discord icon).
